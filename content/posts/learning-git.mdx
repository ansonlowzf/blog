## Basic

```terminal
//To check git version or whether git has been installed in local machine
git -- version

// Update latest git version in windows 10
git update-git-for-windows

// List all global config
git config --global --list (or -l in short)

// Reset or set username in git
git config --global user.name <name>

// Reset or set email in git
git config --global user.email <email>

// Delete username in git
git config --global --unset user.name

// Delete email in git
git config --global --unset user.email

// To tell git start tracking project
git init

// If want to check whether git init is success
ls .git

// Check the file whether on untracked, staging or commit status
git status
```

## Staging

```terminal
//Add a file to staging area
git add <filename>


//Add all file with extension
git add *.html

//Add all modified files (regardless of any extension).
git add . / git add -A
or
git add -A

//Remove file from staging area / unstage.
git rm --cached <filename>

```

## Commit

```terminal
//Commit all staging files to repository with description
git commit -m "<description>"

//
git commit -a -m "<description>"

//Not encourage to just run
git commit

//If your text editor is using Vim, after you run the command; it will enter Vim mode for your to add message.
//After you write your message, you need to press "esc" and type :wq + <enter> to save and exit.
//Or :q! + <enter> to exit without saving.
//It's extra step just to commit and add description.

// Amend commit description
git commit --amend
```

## Git Branch

```terminal
// To create a branch
git branch <branchName>

// To list all branches
git branch

// To delete branch after merging
git branch -d <branchName>

// To delete branch you no longer needed (before merging)
git branch -D <branchName>

// To delete branch in remote repository
git push origin -d <branchName>
```

## Git Checkout

```terminal
// Switch to the branch
git checkout <branchName>

// Switch to master branch
git checkout master

// Create the branch and switch to it
git checkout -b <branchName>

// Swicth to the version of code you want
git checkout <hashId>

// Roll back to previous commit
git checkout -- . (dash dash space dot)

```

## Git Merge

```terminal
// Switch to master branch before merging
git merge <branchName>

// Merge and only include the latest commit log
git merge --squash <branchName>
```

## Git Push

```terminal
// Push latest changes to master
git push origin master

// Push to master and and set up remote branch "master" from "origin"
git push -u origin master

// Push to the origin if you set up upstream before
git push

// Push to the remote branch
git push <networkAddress> <branchName>
```

## Git Pull

```terminal
// Pull the latest changes from the master
git pull origin master

// Set up upstream
git pull -u origin master

// Pull the latest changes from a branch to local branch
git pull <networkAddress> <branchName>
```

## Git Ignore

```terminal
//Tell git to ignore some files which no need to be tracked
Example:-
node_module
```

## Git Clone

```terminal
// Clone from a remote repository to a local machine
// cd to the folder you want to clone
git clone <networkAddress>

// clone and giving it a new name
git clone <networkAddress> <newName>
```

## Git Remote

```terminal
// List remote repository associate with your project
git remote

// Add remote
git remote add origin <remoteNetworkAddress>
also can
git remote add origin 2 <remoteNetworkAddress>

// Remove remote repository from your project
git remote remove origin

// List remote network address associate with your project
git remote -v

// Set remote network address to your project
git remote set-url origin <networkAddress>

// Add a new remote repository to the current repository
git remote add <remoteName> <networkAddress>
```

## Git Fetch

```terminal
// Fetch latest changes from remote repo to local
git fetch origin

// Fetch all lastest changes from all remote branches to local
```

## Git Log

```terminal
// Show logs of our commit
git log

// To see last 2 commits log
git log -2

// To show the commits log in one line
git log --oneline

// To show last 2 commit logs in one line
git log --oneline -2

// To show commit log before the date
git log --before="2020-05-31"

// To show the commit log after the date
git log --after="2020-05-31"

// To show the commit log by the author
git log --author="author name"

// To show log since the date
git log --since="yesterday"

// To show log since some days ago
git log --since="2 weeks ago"

```
