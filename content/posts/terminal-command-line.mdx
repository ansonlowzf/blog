Document command Line

1. pwd = present working directory
1. cd = change directory
1. cd .. = change directory up one level
1. ls = list
1. ls -l = list detail
1. ls -a = list all files include hidden file
1. ls -la = list detail for all files include hidden file
1. mkdir = make directory
1. touch = create file
1. rmdir = remove directory
1. rm "filename" = remove a file
1. rm -rf "filename" = remove a file recursively, avoid asking for your permission to remove every files
1. mv = move a file
1. mv "oldName" "newName" = Rename a file
1. cp = copy a file
